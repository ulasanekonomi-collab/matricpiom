# PIOM Analyzer  
**Power–Institution–Outcome Map**

PIOM Analyzer adalah alat bantu analisis ekonomi politik berbasis kerangka:

**Power → Institution → Incentive → Behavior → Outcome → Design**

- Aplikasi ini dikembangkan untuk membantu:
- Pembelajaran ekonomi politik dan kelembagaan
- Praktisi dalam analisis kebijakan
- Perusahaan dalam membaca struktur organisasi
- Wirausahawan dalam memahami lingkungan bisnis
